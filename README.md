# FilmFinder TMDb Rework

## How to run
1. Open `config.js`
2. Paste your TMDb Read Access Token
3. Open the folder in VS Code
4. Start **Live Server** on `index.html`

## Pages
- `index.html` — Home page with trending slider and content sections
- `films.html` — Movie catalog with search, genre, year, sorting and pagination
- `film_details.html` — Movie details, trailer, favorites, rating and similar movies
- `profile.html` — Local profile with favorites and ratings
- `about.html` — About page

## Notes
- Favorites and ratings are stored in `localStorage`
- Data is loaded from TMDb API
