About FilmFinder

Your personal guide to the world of movies.

FilmFinder is a student portfolio project focused on movie discovery, personal collections and a clean browsing experience built with HTML, CSS and Vanilla JavaScript.

🎯

Mission

Make finding interesting movies simple through clear navigation, practical filters and handpicked sections like Popular, Upcoming and Top Rated.

❤️

Personalization

Users can save favorite titles, leave their own rating and keep everything together in a lightweight local profile.

🎬

Movie exploration

Detail pages collect the essential information: poster, overview, release date, genres, trailer and similar titles.

Main features

  • Interactive banner with trending movies
  • Movie catalog with search, genre, year and sorting
  • Detail page with trailer and similar recommendations
  • Favorites and ratings in a local profile
  • Responsive dark UI inspired by streaming platforms

Tech stack

HTML5 CSS3 Vanilla JavaScript TMDb API localStorage
